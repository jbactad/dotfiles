# dotfiles

Personal dotfiles for macOS and Linux (Ubuntu, Fedora). Configurations are symlinked from this repository into the home directory.

## What's Included

- **Tmux** - Terminal multiplexer config based on [gpakosz/.tmux](https://github.com/gpakosz/.tmux) with local overrides
- **Neovim** - Full IDE-like setup with [coc.nvim](https://github.com/neoclide/coc.nvim), language servers, snippets, and session management
- **Vim** - Lighter configuration sharing the same base settings as Neovim
- **Shell** - Zsh and Bash profile with aliases, PATH setup, and cross-platform clipboard support
- **IdeaVim** - Vim bindings for JetBrains IDEs
- **VsVim** - Vim bindings for Visual Studio
- **Git** - Global gitignore
- **Fonts** - Fira Code Nerd Font (in `third_party/`)

## Prerequisites

- git
- curl
- zsh
- tmux
- vim

For testing: docker

## Installation

```sh
git clone https://github.com/jbactad/dotfiles.git
cd dotfiles
./bin/install.sh
```

### Platform-specific prerequisites

**Debian/Ubuntu:**
```sh
sudo apt-get install git curl zsh tmux vim -y
```

**Fedora:**
```sh
sudo dnf install git curl zsh tmux vim -y
```

**macOS:**
```sh
brew install curl zsh git tmux vim
```

## Testing

Test installation in isolated Docker containers (Ubuntu and Fedora):

```sh
./bin/test.sh
```

## How It Works

`bin/install.sh` creates symlinks from this repository to `~/`:

| Source | Target |
|--------|--------|
| `tmux.conf` | `~/.tmux.conf` |
| `tmux.conf.local` | `~/.tmux.conf.local` |
| `profile` | `~/.profile` |
| `zprofile` | `~/.zprofile` |
| `gitignore` | `~/.gitignore` |
| `ideavimrc` | `~/.ideavimrc` |
| `vsvimrc` | `~/.vsvimrc` |

Vim and Neovim rc files are generated by their respective install scripts (`bin/install-vim.sh`, `bin/install-nvim.sh`) and source configs from the `vimrc/` and `nvimrc/` directories.

Local overrides (not tracked in git) are supported via `.local` file variants — for example, `~/.zprofile.local` is sourced if it exists.
